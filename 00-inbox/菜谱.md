# 红烧肉
葱姜蒜，干辣椒，大料（八角），五香粉，酱油，老抽
- 带皮五花肉（肥肉不要太多的）切1到1.5厘米见方，
- 肉块冷水下锅焯一下（水开5分钟）一边焯一边捞去浮沫，
- 捞出肉控干水分，
- 起锅烧油，油温不要太高炒肉块，
- 炒至表面有点发焦，
- 放入葱姜蒜辣椒大料（五香粉）接着炒1分钟左右，
- 然后放酱油，老抽（一点），翻炒一下，
- 加水没过肉，加盐，
- 炖半小时左右（大火十分钟小火二十分钟，开盖大火收汁）

# 青椒肉丝
青椒，鸡肉切丝（肉丝切粗一点像筷子粗细）
小火，油温不用太高下肉丝炒到变色，
放五香粉，酱油，
大火，
放青椒丝炒到青椒丝看着发软，
放盐，味精

# 角瓜香肠
角瓜和香肠切片，不要太厚
葱蒜五香粉爆锅
煎一下香肠
放角瓜炒软
放酱油一点点，盐一点点，味精一点
淀粉一点化水，勾芡
